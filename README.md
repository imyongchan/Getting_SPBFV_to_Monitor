📌 Project Overview

고혈압, 동맥경화, 당뇨병, 비만 등 다양한 요인으로 인한 심뇌혈관 질환은 고령화와 함께 환자 수가 지속적으로 증가하고 있습니다.
하지만 대부분의 질환은 초기 이상 증상이 뚜렷하지 않아 조기 발견과 지속적인 모니터링이 매우 중요합니다.

본 프로젝트는:

시간과 장소의 제약 없이 사용 가능한

PPG 센서(광용적맥파 측정 센서) 와

소형 아두이노 기반 웨어러블 시스템을 활용하여

👉 공간말초혈류속도(SPBFV) 를 측정하는
손목 착용형 스마트 혈류속도 측정기를 개발하는 것을 목표로 합니다.
