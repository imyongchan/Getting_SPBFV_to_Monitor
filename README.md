## 📌 프로젝트 개요

고혈압, 동맥경화, 당뇨병, 비만 등 다양한 요인으로 인한 심뇌혈관 질환은
고령화와 함께 지속적으로 증가하고 있습니다.

하지만 대부분의 심뇌혈관 질환은 초기 증상이 뚜렷하지 않아  
**조기 발견과 지속적인 모니터링이 매우 중요합니다.**

본 프로젝트는 시간과 장소의 제약 없이 사용할 수 있는  
**손목 착용형 스마트 혈류속도 측정기**를 개발하는 것을 목표로 합니다.

PPG 센서(광용적맥파 센서) 2개와 아두이노 기반 시스템을 활용하여  
새로운 생체지표인 **공간말초혈류속도(SPBFV)** 를 측정합니다.
